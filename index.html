HTML CSS

En la segunda semana se aprendio sobre el HTML "HyperText Markup Language" , pues se trata de un lenguaje que define y estructura el contenido de las paginas web, pues su uso se facilita por el uso de etiquetas , ademas de ser parte del estandar SGML , posteriormente se trato el XML como el estandar a utilizar para almacenar e intercambiar informacion estructurada y el DOM, para leer y manipular el contenido, estructura y estilos de la pagina.

03

LIBRERIAS

En la tercera semana se trato el CSS o Cascading Style Sheets, ademas de las librerias, pues se trata de un lenguaje usado en la presentacion del contenido de una pagina, ademas se trato el desarrollo de estilos con CLASS, pues ello permite definir estilos reutilizables, los estilos con ID para los estilos no reutilizables, y el Flexbox para generar elementos flexibles que se adapten a su contenedor.

JAVASCRIPT Y ECMASCRIPT

En la cuarta semana se trato JavaScript, pues se trata de un lenguaje script dinamico, multi-paradigma y orientado a objetos, de la misma manera se trato al NODE.js, que refiere a un entorno de ejecucion de codigo JavaScript en el servidor, por ultimo se trato sobre las variables, estructuras, funciones y procedimientos.

REACT

En la quinta semana se abarco el tema de React, una biblioteca JavaScript que se utiliza para crear SPA (aplicaciones de una sola pagina) ademas de componentes reutilizables, luego se trato sobre la creacion del DOM Virtual y de las Dependencias como Babel JS, ESLint, Sass, etc.

Conclusión

En el desarrollo de este proyecto, se exploraron diversos aspectos de las tecnologías web y su aplicación en el desarrollo de software. Cada tema abordado proporcionó una comprensión más profunda de los fundamentos tecnológicos.

La experiencia adquirida al trabajar con React, enrutadores, estilos, Next.js y otras tecnologías web es valiosa. Se identificaron desafíos y se aplicaron soluciones, contribuyendo al crecimiento profesional.
